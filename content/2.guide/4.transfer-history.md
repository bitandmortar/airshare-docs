# Transfer History

AirShare keeps a detailed record of all your file transfers, making it easy to track what you've sent and received.

## Accessing History

Click the **History** tab in the main AirShare window to view all past transfers.

## Understanding the History View

### What You See

Each transfer entry shows:

- **File icon**: Type of file(s) transferred
- **File name(s)**: What was sent/received
- **Direction icon**: ↑ Sent or ↓ Received
- **Other device**: Who you sent to or received from
- **Date & time**: When the transfer occurred
- **File size**: Total size of transfer
- **Status badge**: Completed, Failed, or Cancelled

### Color Coding

- **Green**: Successfully completed
- **Red**: Failed or error
- **Gray**: Cancelled
- **Blue**: In progress

## Filtering History

### Filter by Direction

1. Click **Filter** button
2. Select:
   - **All transfers** (default)
   - **Sent only**
   - **Received only**

### Filter by Status

1. Click **Filter** button
2. Select:
   - **All statuses** (default)
   - **Completed**
   - **Failed**
   - **Cancelled**

### Filter by Date Range

1. Click **Filter** button → **Date Range**
2. Choose:
   - **Today**
   - **Last 7 days**
   - **Last 30 days**
   - **Last 90 days**
   - **All time**
   - **Custom range** (select start and end dates)

### Filter by Device

1. Click **Filter** button → **Device**
2. Select a specific device to see only transfers with that device

## Searching History

### Quick Search

Use the search bar at the top of History:

1. Type any part of:
   - File name
   - Device name
   - Date
2. Results filter in real-time

### Advanced Search

Click **Advanced Search** for more options:

- **File name contains**: Search file names
- **Device name contains**: Search by sender/recipient
- **File size**: Filter by size range
- **File type**: Filter by extension (.jpg, .pdf, etc.)
- **Date range**: Specific date range

::u-callout
---
icon: i-lucide-search
color: blue
---
**Search tip**: Use wildcards (*) for broader searches, e.g., `*.jpg` finds all JPG transfers.
::

## Sorting History

Click column headers to sort:

- **Date**: Newest or oldest first
- **File name**: Alphabetical
- **File size**: Largest or smallest first
- **Device name**: Alphabetical
- **Status**: Group by status

Click again to reverse sort order.

## Transfer Details

### Viewing Full Details

Click on any transfer entry to see complete information:

#### General Information
- Complete file list (if multiple files)
- Full device name
- Exact transfer date and time
- Duration of transfer
- Average transfer speed

#### File Information
- Individual file names and sizes
- File types
- Total size
- Save location (for received files)

#### Transfer Statistics
- Start time
- End time
- Total duration
- Average speed (MB/s)
- Protocol used (QUIC)

### Actions Available

From transfer details, you can:

- **Open file**: Open the transferred file (if it still exists)
- **Show in folder**: Open the folder containing the file
- **Resend**: Send the same file(s) again
- **Copy file path**: Copy file location to clipboard
- **Delete from history**: Remove this entry

## Managing History

### Delete Individual Entries

1. Click on a transfer entry
2. Click **Delete** or press **Delete** key
3. Confirm deletion

::u-callout
---
icon: i-lucide-info
---
**Note**: Deleting from history doesn't delete the actual files - only the history record.
::

### Delete Multiple Entries

1. Enable **Select mode** (checkbox icon)
2. Check boxes next to entries to delete
3. Click **Delete Selected**
4. Confirm deletion

### Clear All History

1. Click **Settings** → **Storage** → **History**
2. Click **"Clear All History"**
3. Type "DELETE" to confirm
4. All history records are permanently removed

::u-callout
---
icon: i-lucide-alert-triangle
color: red
---
**Warning**: Clearing history cannot be undone. The action is permanent.
::

## Export History

### Export to CSV

Export your transfer history for record-keeping:

1. Click **Export** button in History
2. Choose **CSV format**
3. Select date range to export
4. Choose save location
5. Open in Excel, Google Sheets, etc.

CSV includes:
- Date/time
- Direction (sent/received)
- File name(s)
- File size
- Device name
- Status
- Duration
- Speed

### Export to PDF

Generate a PDF report:

1. Click **Export** button in History
2. Choose **PDF format**
3. Select date range
4. Choose save location

PDF includes formatted transfer summary.

## Storage Settings

### History Retention

Configure how long to keep history:

1. Go to **Settings** → **Storage** → **History Retention**
2. Choose:
   - **Keep all history** (default)
   - **Auto-delete after 30 days**
   - **Auto-delete after 90 days**
   - **Auto-delete after 1 year**

Old entries are automatically removed based on your setting.

### Database Size

View how much space history uses:

1. Go to **Settings** → **Storage**
2. See **"History database size"**
3. Typical size: 1-10 MB for months of transfers

### Optimize Database

If history becomes slow:

1. Go to **Settings** → **Storage**
2. Click **"Optimize Database"**
3. Vacuums and compacts the database
4. Improves query speed

## Privacy Considerations

### What's Stored

History stores:
- ✅ File names and sizes
- ✅ Device names
- ✅ Transfer dates/times
- ✅ Transfer status
- ✅ File paths (on your system)

### What's NOT Stored

History does NOT store:
- ❌ File contents
- ❌ IP addresses
- ❌ Network details
- ❌ User account information

### Disable History

To not save any history:

1. Go to **Settings** → **Privacy** → **History**
2. Disable **"Save transfer history"**
3. No transfers will be logged

::u-callout
---
icon: i-lucide-eye-off
---
**Privacy mode**: Disabling history means no records are kept, but you also can't track past transfers.
::

## Troubleshooting History Issues

### History Not Updating

**Problem**: New transfers don't appear in History

**Solutions**:
- Check history is enabled: Settings → Privacy → History
- Restart AirShare
- Check database isn't corrupted: Settings → Storage → Optimize Database

### Missing Entries

**Problem**: Some transfers are missing from History

**Solutions**:
- Check date range filter isn't too restrictive
- Check if auto-delete removed old entries
- Verify search filters aren't active

### Can't Open Files from History

**Problem**: "File not found" when trying to open from History

**Solutions**:
- File may have been moved or deleted after transfer
- Check original save location
- Use "Show in Folder" to navigate to location

### Slow History Loading

**Problem**: History takes long to load or search

**Solutions**:
- Click Settings → Storage → **Optimize Database**
- Consider clearing old entries
- Enable auto-delete for old transfers

## History FAQs

**Q: Does history sync between devices?**  
A: Not currently. Each device maintains its own local history.

**Q: How much storage does history use?**  
A: Very little - typically 1-10 MB even for thousands of transfers.

**Q: Can I recover deleted history?**  
A: No, history deletion is permanent. Export regularly for backups.

**Q: Is history encrypted?**  
A: History is stored in a local SQLite database on your device.

**Q: Can I import history from another device?**  
A: Not currently, but planned for future releases.

**Q: What happens to history if I uninstall?**  
A: History database is deleted unless you backup the app data folder first.
