# Settings Reference

This comprehensive guide covers every setting available in AirShare, helping you customize your experience.

::u-callout
---
icon: i-lucide-info
---
Access settings by clicking the **gear icon** (⚙️) in the main window or from the system tray menu.
::

## General Settings

### Device & Identity

#### Device Name
- **What it does**: Sets how your device appears to others in their Radar
- **Default**: Your computer's hostname
- **Recommendation**: Use a descriptive, recognizable name

```
Examples:
✅ "Sarah's MacBook Pro"
✅ "Office Desktop - Marketing"
✅ "John's Windows Laptop"
❌ "localhost"
❌ "Device-12345"
```

### Language

#### Interface Language
- **Options**: English, Croatian, German, French, Italian, Spanish
- **Default**: System language (if supported), otherwise English
- **Effect**: Changes all UI text immediately

::u-callout
---
icon: i-lucide-globe
color: blue
---
**Contribute**: Want to add your language? Visit our [GitHub repository](https://github.com/airsharehq/airshare).
::

### Downloads

#### Default Download Location
- **What it does**: Where received files are saved
- **Default**: 
  - Windows: `C:\Users\YourName\Downloads`
  - macOS: `~/Downloads`
  - Linux: `~/Downloads`
- **Recommendation**: Choose a location with plenty of free space

#### Ask Where to Save Each Transfer
- **Enabled**: Choose location for each transfer
- **Disabled**: Always use default location
- **Default**: Disabled
- **Recommendation**: Enable if you organize files into specific folders

#### Open Folder After Receiving
- **Enabled**: Automatically opens folder when transfer completes
- **Disabled**: Silent completion
- **Default**: Enabled
- **Recommendation**: Keep enabled for convenience

## Notifications

### Transfer Notifications

#### Incoming Transfer Requests
- **Enabled**: Notify when someone wants to send you files
- **Default**: Enabled
- **Required**: Highly recommended to know about incoming files

#### Transfer Started
- **Enabled**: Notify when a transfer begins
- **Default**: Enabled
- **Use case**: Helpful for tracking outgoing transfers

#### Transfer Complete
- **Enabled**: Notify when transfers finish successfully
- **Default**: Enabled
- **Recommendation**: Keep enabled

#### Transfer Failed
- **Enabled**: Notify when transfers fail or are cancelled
- **Default**: Enabled
- **Recommendation**: Keep enabled to catch issues

### Sound Alerts

#### Play Sound for Notifications
- **Enabled**: Plays system sound with notifications
- **Default**: Enabled
- **Recommendation**: Disable if you work in quiet environments

#### Sound Volume
- **Range**: 0-100%
- **Default**: 50%
- **Adjustment**: Use slider to set comfortable volume

## Privacy & Security

### Visibility

#### Visible to Others
- **Enabled**: Your device appears in others' Radars
- **Disabled**: Invisible mode - you can send but not receive
- **Default**: Enabled

::u-callout
---
icon: i-lucide-eye-off
---
**Invisible mode**: Useful when you want to send files but not receive any.
::

### Transfer Permissions

#### Transfer Mode
- **Ask every time**: Manual approval for each transfer (recommended)
- **Auto-accept from known devices**: Automatically accept from trusted senders
- **Auto-accept all**: Accept everything (NOT recommended)
- **Block all**: Reject all incoming transfers
- **Default**: Ask every time

::u-callout
---
icon: i-lucide-shield-check
color: green
---
**Security tip**: Use "Ask every time" on public or work networks.
::

### Known Devices

#### Manage Trusted Devices
- View all devices that have sent you files
- Mark devices as "Trusted" for auto-accept
- Block specific devices permanently

**Actions**:
- **Trust**: Auto-accept transfers from this device (if auto-accept enabled)
- **Block**: Always reject transfers from this device
- **Remove**: Forget this device

### File Type Restrictions

#### Block Executable Files
- **Enabled**: Automatically rejects .exe, .bat, .sh, .app, .dmg, .deb, etc.
- **Default**: Disabled
- **Recommendation**: Enable for extra security

#### Allowed File Types (Custom)
- **All types**: Accept any file type (default)
- **Custom filter**: Only accept specific extensions

```
Example custom filter:
.jpg, .png, .gif, .pdf, .docx, .xlsx, .txt, .zip
```

### History Privacy

#### Save Transfer History
- **Enabled**: Keep records of all transfers (default)
- **Disabled**: Don't save any history (privacy mode)

::u-callout
---
icon: i-lucide-alert-triangle
color: amber
---
**Note**: Disabling history means you can't track past transfers or view statistics.
::

## Network Settings

### Discovery

#### Enable Device Discovery
- **Enabled**: Allow AirShare to find and be found by other devices
- **Default**: Enabled
- **Required**: Must be enabled to use AirShare

#### Discovery Interval
- **Range**: 1-30 seconds
- **Default**: 5 seconds
- **Fast (1-5s)**: More responsive, slightly more battery usage
- **Slow (10-30s)**: Better battery life, slower to find devices
- **Recommendation**: 5 seconds for best balance

#### Peer Timeout
- **Range**: 15-120 seconds
- **Default**: 30 seconds
- **What it does**: How long to wait before removing inactive devices from Radar

### Ports (Advanced)

#### QUIC Port Range
- **Default**: 59875-59925 (50 ports)
- **What it does**: Ports used for file transfers
- **Recommendation**: Don't change unless you know what you're doing

::u-callout
---
icon: i-lucide-alert-circle
color: red
---
**Warning**: Incorrect port settings can break device discovery. Only change if required by your network.
::

#### Custom Port
- **Enabled**: Use a specific port instead of dynamic range
- **Default**: Disabled
- **Use case**: Required firewall configuration, port forwarding

## Appearance

### Theme

#### Color Scheme
- **Light**: Traditional light theme
- **Dark**: Dark mode (easy on the eyes)
- **System**: Follows your OS theme preference
- **Default**: System

#### Accent Color
- **Options**: Blue, Purple, Green, Orange, Red, Pink
- **Default**: Blue
- **Effect**: Changes highlight colors throughout the app

### Window Behavior

#### Minimize to System Tray
- **Enabled**: Closing window minimizes to tray
- **Disabled**: Closing window quits app
- **Default**: Enabled
- **Recommendation**: Keep enabled to stay ready for transfers

#### Start Minimized
- **Enabled**: Launch in system tray (window hidden)
- **Disabled**: Show window on launch
- **Default**: Disabled

#### Always on Top
- **Enabled**: AirShare window stays above other windows
- **Disabled**: Normal window behavior
- **Default**: Disabled

#### Show Dock Icon (macOS only)
- **Enabled**: Show in macOS dock
- **Disabled**: Only in menu bar
- **Default**: Enabled

## Storage & Database

### Transfer History

#### History Retention
- **Keep all**: Never delete history automatically
- **30 days**: Auto-delete entries older than 30 days
- **90 days**: Auto-delete entries older than 90 days
- **1 year**: Auto-delete entries older than 1 year
- **Default**: Keep all

#### Database Management

**Database Size**: Shows current history database size

**Optimize Database**: 
- Compacts and vacuums the SQLite database
- Run if history becomes slow
- Safe to use anytime

**Clear All History**:
- Permanently deletes all history records
- Cannot be undone
- Requires typing "DELETE" to confirm

### Cache

#### Cache Size Limit
- **Range**: 100 MB - 10 GB
- **Default**: 500 MB
- **What it does**: Temporary storage for transfers in progress

**Clear Cache**:
- Removes temporary transfer files
- Safe to use when no transfers are active
- Frees up disk space

## Performance

### Transfers

#### Maximum Concurrent Transfers
- **Range**: 1-5
- **Default**: 1 (queue mode)
- **Pro**: Up to 5 simultaneous transfers
- **Impact**: More concurrent transfers may reduce individual speeds

#### Transfer Buffer Size
- **Options**: Small (32 KB), Medium (64 KB), Large (128 KB), Huge (256 KB)
- **Default**: Medium (64 KB)
- **Large files**: Use "Large" or "Huge" for files over 1 GB
- **Small files**: Use "Small" or "Medium"

#### Chunk Size
- **Range**: 1-16 MB
- **Default**: 4 MB
- **Impact**: Larger chunks = fewer protocol overhead for large files

::u-callout
---
icon: i-lucide-zap
color: yellow
---
**Performance tip**: For gigabit networks and large files, use Large buffer + 8 MB chunks.
::

### Network Optimization

#### Enable TCP Congestion Control
- **Enabled**: Better for congested networks (default)
- **Disabled**: Maximum speed on fast, uncongested networks

#### Keep-Alive Interval
- **Range**: 5-60 seconds
- **Default**: 15 seconds
- **What it does**: How often to check connection is alive

## System

### Startup

#### Launch on System Startup
- **Enabled**: Auto-start AirShare when computer boots
- **Default**: Disabled
- **Recommendation**: Enable to always be ready for transfers

::u-callout
---
icon: i-lucide-info
---

::

#### Start with Window Hidden
- **Enabled**: Launch to tray only
- **Disabled**: Show window on startup
- **Default**: Disabled
- **Requires**: "Launch on startup" must be enabled

### Updates

#### Check for Updates Automatically
- **Enabled**: Check for new versions on startup
- **Default**: Enabled
- **Recommendation**: Keep enabled for security updates

#### Download Updates Automatically
- **Enabled**: Auto-download updates when available
- **Disabled**: Notify only, manual download
- **Default**: Disabled

#### Install Updates Automatically
- **Enabled**: Install and restart when ready
- **Disabled**: Manual installation
- **Default**: Disabled

::u-callout
---
icon: i-lucide-shield-check
color: green
---
**Security**: All updates are cryptographically signed. You'll be notified before any restart.
::

**Check Now**: Manually check for updates anytime

**Update Channel**:
- **Stable**: Official releases only (recommended)
- **Beta**: Early access to new features (may have bugs)

## Advanced

### Logging

#### Enable Debug Logging
- **Enabled**: Save detailed logs for troubleshooting
- **Default**: Disabled
- **Location**:

  - macOS: `~/Library/Logs/airshare`
  - Linux: `~/.local/share/airshare/logs`

::u-callout
---
icon: i-lucide-bug
color: blue

**Troubleshooting**: Enable debug logs when reporting bugs or issues.
::

- **Error**: Only errors
- **Warn**: Warnings and errors
- **Info**: General information (default)
- **Debug**: Detailed debug information
- **Trace**: Everything (very verbose)

**View Logs**: Opens log folder

## Resetting Settings

### Reset to Defaults

1. Go to **Settings** → **Advanced**
2. Scroll to bottom
3. Click **"Reset All Settings"**
4. Confirm reset
5. All settings return to defaults
6. History and license info are preserved

### Complete Reset

To completely reset AirShare including history:

1. Quit AirShare
2. Delete app data folder:
   - Windows: `%APPDATA%\io.github.jaksatomovic.airshare`
   - macOS: `~/Library/Application Support/io.github.jaksatomovic.airshare`
   - Linux: `~/.config/io.github.jaksatomovic.airshare`
3. Relaunch AirShare
4. Fresh installation state
