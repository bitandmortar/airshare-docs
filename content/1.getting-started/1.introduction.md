# Introduction to AirShare

AirShare is a **cross-platform file sharing application** that lets you transfer files instantly between devices on the same local network - no internet connection required!

## What is AirShare?

Think of AirShare as **AirDrop for everyone**. Whether you're using Windows, macOS, or Linux, AirShare makes sharing files as simple as drag-and-drop.

### Key Features

- **🚀 Lightning Fast**: Transfer files at your full network speed using the QUIC protocol
- **🔒 Secure**: TLS 1.3 encryption keeps your files safe
- **🌐 Cross-Platform**: Works on Windows, macOS, and Linux
- **📡 Auto-Discovery**: Automatically finds nearby devices running AirShare
- **💾 Transfer History**: Keep track of all your sent and received files
- **🎨 Beautiful UI**: Modern interface with dark mode support
- **🌍 Multilingual**: Available in 6 languages

## How Does It Work?

AirShare uses **local network discovery** to find other devices running the app. When you want to send a file:

1. Open AirShare on both devices
2. The devices automatically discover each other
3. Drag and drop your files onto a recipient
4. They accept the transfer
5. Files are transferred directly between devices at full speed

## System Requirements

### Minimum Requirements
- **Operating System**: 
  - Windows 10 or later
  - macOS 10.15 (Catalina) or later
  - Linux (modern distributions with glibc 2.31+)
- **Network**: Wi-Fi or Ethernet connection
- **RAM**: 256 MB available memory
- **Disk Space**: 100 MB for installation

### Recommended
- **Network**: Gigabit Ethernet or 5 GHz Wi-Fi for best transfer speeds
- **RAM**: 512 MB available memory

## What Makes AirShare Different?

### vs. AirDrop
- ✅ Works on **all platforms**, not just Apple devices
- ✅ Transfer history with searchable database
- ✅ Customizable settings and preferences

### vs. Cloud Storage (Google Drive, Dropbox)
- ✅ No internet connection required
- ✅ Instant transfers at full local network speed
- ✅ No file size limits (in Pro version)
- ✅ Complete privacy - files stay on your network

### vs. Email/USB Drives
- ✅ No file size limits for attachments
- ✅ Faster than copying to USB
- ✅ No physical connection needed
- ✅ Send to multiple devices simultaneously

## Privacy & Security

Your privacy is our top priority:

- 🔒 **End-to-end encryption**: All transfers use TLS 1.3
- 🏠 **Local network only**: Files never leave your network
- ✅ **Hash verification**: SHA-256 ensures file integrity
- 🔐 **No tracking**: We don't collect usage data or analytics
- 📝 **Open source**: Code is publicly auditable
