# Quick Start Guide

Welcome! Let's get you sending files in under 2 minutes. 🚀

::u-callout
---
icon: i-lucide-info
---
**Before you start**: Make sure AirShare is installed on **at least two devices** on the same network.
::

## Step 1: Launch AirShare

Open AirShare on both devices:

- **Windows**: Find it in the Start Menu
- **macOS**: Check Applications folder or use Spotlight (⌘ + Space)
- **Linux**: Find it in your app menu or run `airshare`

::u-callout
---
icon: i-lucide-lightbulb
color: blue
---
**Tip**: AirShare will appear in your system tray. You can minimize the window and still send/receive files!
::

## Step 2: Verify Devices are Discovered

Once AirShare is running on both devices:

1. Look at the **Radar** tab (should be open by default)
2. Within a few seconds, you should see other devices appear as cards
3. Each card shows:
   - Device name
   - Operating system icon
   - Avatar/initial

::u-callout
---
icon: i-lucide-alert-circle
color: amber
---
**Not seeing other devices?** Jump to [Troubleshooting Discovery Issues](/troubleshooting/common-issues#discovery-issues).
::

## Step 3: Send Your First File

### Method 1: Drag and Drop (Recommended)

1. **Drag files** from your file explorer onto a device card in the Radar
2. A **confirmation dialog** will appear showing:
   - File names and sizes
   - Total transfer size
   - Recipient device name
3. Click **"Send"**
4. Wait for the recipient to accept

### Method 2: Click to Select

1. Click on a device card in the Radar
2. Click **"Select Files"** in the send dialog
3. Choose files from the file picker
4. Click **"Send"**

## Step 4: Accept the Transfer (Recipient)

On the receiving device:

1. A **notification** will appear (both in-app and system notification)
2. The **receive confirmation dialog** opens automatically showing:
   - Sender's name
   - List of files being sent
   - Total size
3. Click **"Accept"** to receive
4. Or click **"Decline"** to reject

::u-callout
---
icon: i-lucide-shield-check
color: green
---
**Security**: Always verify the sender's name before accepting files.
::

## Step 5: Monitor Transfer Progress

During the transfer:

- **Progress bar** shows transfer percentage
- **Speed indicator** shows current transfer rate (MB/s)
- **Time remaining** shows estimated completion time
- **Cancel button** available if needed

::u-callout
---
icon: i-lucide-info
---
**Transfer speeds** depend on your network. Expect:
- Gigabit Ethernet: 100+ MB/s
- Wi-Fi 6 (5GHz): 50-100 MB/s
- Wi-Fi 5 (5GHz): 20-50 MB/s
::

## Step 6: Transfer Complete!

Once finished:

- ✅ You'll see a **"Transfer Complete"** notification
- 📁 Files are automatically saved to your Downloads folder (or custom location)
- 📊 Transfer is logged in the **History** tab

## View Transfer History

Click on the **History** tab to see:

- All sent and received files
- Transfer dates and times
- File sizes
- Sender/recipient information
- Status (completed, cancelled, failed)

You can also:
- 🔍 **Search** for specific transfers
- 🗑️ **Delete** history entries
- 📂 **Open** the file location directly

## Common First-Time Tasks

### Change Download Location

1. Click **Settings** (gear icon)
2. Go to **General** → **Downloads**
3. Click **"Change Download Folder"**
4. Select your preferred location

### Change Language

1. Click **Settings** (gear icon)
2. Go to **General** → **Language**
3. Select from: English, Croatian, German, French, Italian, Spanish

### Enable Dark Mode

1. Click **Settings** (gear icon)
2. Go to **Appearance** → **Theme**
3. Choose: **Light**, **Dark**, or **System**

### Set Launch on Startup

1. Click **Settings** (gear icon)
2. Go to **General** → **System**
3. Enable **"Launch on startup"**

## Understanding the Interface

### Main Tabs

- **📡 Radar**: See nearby devices and send files
- **📜 History**: View all past transfers
- **⚙️ Settings**: Configure AirShare preferences

### System Tray Icon

The system tray icon shows:
- **Green dot**: Ready to transfer
- **Pulsing animation**: Transfer in progress

Right-click the tray icon for quick actions:
- Show/Hide window
- Quit AirShare

## Tips for Best Experience

::u-callout
---
icon: i-lucide-zap
color: yellow
---
### Pro Tips

1. **Use Ethernet for large files**: Wired connections are faster and more stable
2. **Connect to 5GHz Wi-Fi**: 2.4GHz is slower and more congested
3. **Keep AirShare running**: It uses minimal resources and enables instant transfers
4. **Accept the EULA**: Required for transfers to work properly
::
